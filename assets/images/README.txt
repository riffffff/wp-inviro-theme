PLACEHOLDER IMAGES NEEDED
==========================

This folder should contain placeholder images for the theme. You can add:

1. hero-placeholder.jpg (1920x800px) - For hero slider when no posts are found
2. product-placeholder.jpg (400x400px) - For products without featured images
3. branch-placeholder.jpg (300x200px) - For branch locations without images
4. avatar-placeholder.jpg (100x100px) - For testimonials without profile pictures

You can use any image editing software or download free placeholder images from:
- https://placeholder.com
- https://unsplash.com
- https://pexels.com

Note: These are optional - the theme will work without them, but images enhance the visual appeal.
